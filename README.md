# Suluu Business

Автономное приложение для управления beauty-бизнесом. Учёт записей, мастеров, клиентов, услуг и финансов.

## Архитектура

```
┌─────────────────────┐     fetch()     ┌─────────────────────┐
│   Фронтенд (SPA)    │ ──────────────► │  Google Apps Script  │
│   GitHub Pages       │ ◄────────────── │    (Web App)         │
│   HTML/CSS/JS        │     JSON        │                      │
└─────────────────────┘                 └──────────┬───────────┘
                                                    │
                                          SpreadsheetApp
                                                    │
                                        ┌───────────▼──────────┐
                                        │   Google Таблица      │
                                        │   (База данных)       │
                                        └──────────────────────┘
```

- **Фронтенд**: HTML + Vanilla JS + Tailwind CSS + morphdom. Хостинг на GitHub Pages.
- **Бэкенд**: Google Apps Script, деплоится как Web App.
- **БД**: Google Таблица с листами: Settings, Masters, Services, Bookings, Clients, Transactions, Shifts, PriceHistory.

## Функционал

| Модуль | Описание |
|--------|----------|
| 📊 Дашборд | Метрики выручки, записей, конверсии. Графики за неделю/месяц |

| 💰 Финансы | Кошельки, транзакции, смены кассы |

## Быстрый старт

### 1. Настройка бэкенда (Google Apps Script)

Подробная инструкция: [gas/README.md](gas/README.md)

Краткие шаги:
1. Создайте новую Google Таблицу
2. Откройте **Расширения → Apps Script**
3. Скопируйте все `.gs` файлы из папки `gas/`
4. Деплойте как **Web App** (Выполнять как: Я, Доступ: Все)
5. Скопируйте URL деплоя

### 2. Деплой фронтенда

**Вариант A: GitHub Pages**
1. Загрузите содержимое папки `business/` в GitHub-репозиторий
2. Включите GitHub Pages (Settings → Pages → Branch: main, Folder: / (root))
3. Откройте сайт, введите URL GAS Web App при первом входе

**Вариант B: Локально**
1. Откройте `index.html` через Live Server (VS Code extension) или любой HTTP-сервер
2. Введите URL GAS Web App при первом входе

### 3. Первый вход

1. Откройте приложение
2. Введите URL вашего GAS Web App
3. Придумайте пароль (при первом входе он станет паролем администратора)
4. Готово!
